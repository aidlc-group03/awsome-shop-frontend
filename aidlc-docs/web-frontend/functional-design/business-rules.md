# Web Frontend — 业务规则

## BR-F1: 路由与权限规则

| 规则ID | 规则描述 |
|--------|---------|
| BR-F1.1 | 未登录用户访问任何受保护路由 → 跳转 /login |
| BR-F1.2 | employee 角色访问 /admin/* → 跳转 / |
| BR-F1.3 | admin 角色访问 / → 跳转 /admin |
| BR-F1.4 | Token 存储在 localStorage('token')，用户信息通过 Zustand persist 持久化 |
| BR-F1.5 | 401 响应 → 清除本地状态 → 跳转 /login |

---

## BR-F2: 商品浏览规则 (员工端)

| 规则ID | 规则描述 |
|--------|---------|
| BR-F2.1 | 商品列表仅显示已上架 (status=1) 的商品 |
| BR-F2.2 | 分类标签第一项为"全部"，点击取消分类筛选 |
| BR-F2.3 | 商品卡片展示: 图片、名称、积分价格、已兑换数量 |
| BR-F2.4 | 库存为0的商品显示"已兑完"标记，兑换按钮不可点击 |
| BR-F2.5 | 分页: 每页默认20条，支持无限滚动或传统分页 |

---

## BR-F3: 兑换流程规则 (员工端)

| 规则ID | 规则描述 |
|--------|---------|
| BR-F3.1 | 积分不足时，"立即兑换"按钮 disabled + 提示"积分不足" |
| BR-F3.2 | 兑换确认页显示: 商品信息摘要 + 将扣除的积分数量 + 配送表单 |
| BR-F3.3 | 配送表单必填: 收件人姓名、手机号、配送地址 |
| BR-F3.4 | 手机号格式验证: 11位数字，以1开头 |
| BR-F3.5 | 确认兑换按钮点击后禁用，防止重复提交 |
| BR-F3.6 | 兑换成功后跳转成功页，显示订单号 |
| BR-F3.7 | 兑换失败显示具体原因（积分不足/库存不足/系统繁忙） |

---

## BR-F4: 订单展示规则

| 规则ID | 规则描述 |
|--------|---------|
| BR-F4.1 | 员工只能看到自己的订单 |
| BR-F4.2 | 订单状态使用不同颜色标签: pending=橙色, confirmed=蓝色, shipping=紫色, completed=绿色, cancelled=灰色 |
| BR-F4.3 | 订单列表支持按状态筛选 |
| BR-F4.4 | 订单详情展示完整信息: 商品、积分、配送信息、状态时间线 |

---

## BR-F5: 积分展示规则

| 规则ID | 规则描述 |
|--------|---------|
| BR-F5.1 | 积分中心顶部显示: 当前余额 (大字)、累计获得、累计消费 |
| BR-F5.2 | 交易记录类型颜色: earn/admin_add=绿色(+), spend/admin_deduct=红色(-), refund=蓝色(+) |
| BR-F5.3 | 交易记录支持按类型筛选 |
| BR-F5.4 | 导航栏始终显示当前积分余额 |

---

## BR-F6: 管理端通用规则

| 规则ID | 规则描述 |
|--------|---------|
| BR-F6.1 | 所有管理列表页支持: 搜索、筛选、分页、排序 |
| BR-F6.2 | 删除操作必须弹出确认对话框 |
| BR-F6.3 | 表单提交后，成功则关闭Dialog并刷新列表，失败则显示错误不关闭 |
| BR-F6.4 | 表格空状态显示友好提示 |
| BR-F6.5 | 低库存商品(stock < 10)在列表中标记红色警示 |

---

## BR-F7: 管理端积分操作规则

| 规则ID | 规则描述 |
|--------|---------|
| BR-F7.1 | 发放积分: 选择目标用户 + 输入积分数(正整数) + 必须填写原因 |
| BR-F7.2 | 扣除积分: 选择目标用户 + 输入积分数 + 显示当前余额 + 必须填写原因 |
| BR-F7.3 | 扣除积分数不能超过用户当前余额（前端校验 + 后端校验双重保障） |
| BR-F7.4 | 积分操作成功后刷新用户积分列表 |

---

## BR-F8: 管理端订单操作规则

| 规则ID | 规则描述 |
|--------|---------|
| BR-F8.1 | 状态流转按钮根据当前状态显示: pending→确认, confirmed→发货, shipping→完成 |
| BR-F8.2 | 已完成/已取消的订单无操作按钮 |
| BR-F8.3 | 状态更新前弹出确认对话框 |

---

## BR-F9: 国际化规则

| 规则ID | 规则描述 |
|--------|---------|
| BR-F9.1 | 登录页跟随浏览器语言 |
| BR-F9.2 | 登录后使用用户偏好语言（持久化在 localStorage） |
| BR-F9.3 | 所有用户可见文本均通过 i18n key 引用 |
| BR-F9.4 | 数字格式（积分数量）跟随 locale |

---

## BR-F10: 主题规则

| 规则ID | 规则描述 |
|--------|---------|
